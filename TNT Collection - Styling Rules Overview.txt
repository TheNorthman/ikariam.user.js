TNT Collection - Styling Rules Overview

========================================================================================================================
| CATEGORY             | RESOURCES TABLE                | BUILDINGS TABLE               | COMMON RULES
========================================================================================================================
| Base Styles          | 12px Arial, #fdf7dd bg       | Same                          | Collapsed borders, 4px padding
| Headers              | #faeac6 bg, bold, center     | Same                          | Dynamic category spans
------------------------------------------------------------------------------------------------------------------------
| Column Visibility    | City+Lvl always visible,       | Only shows buildings          | Based on user settings/data
|                      | others toggleable              | that exist                    |
------------------------------------------------------------------------------------------------------------------------
| Value Warnings       | storage_min: #FF000050       | tnt_construction:             | Red/orange tints for alerts
|                      | when < threshold               | #80404050 for upgrading     |
------------------------------------------------------------------------------------------------------------------------
| Storage Thresholds   | Wood/Wine: <100k               | N/A                           | storage_danger: >80% max
|                      | Marble/Crystal/Sulfur: <50k    |                               |
------------------------------------------------------------------------------------------------------------------------
| Production           | Bold font for wood and         | N/A                           | Hover shows additional info
|                      | the tradegood that is produced |                               |
|                      | in this city                   |                               |
------------------------------------------------------------------------------------------------------------------------
| Indicators           | cities, tooltips show          | in tooltips                   |
|                      | 24h production                 |                               |
------------------------------------------------------------------------------------------------------------------------
| Current City         | 2px black border around        | Same                          | No background change, just border
|                      | entire row                     |                               |
------------------------------------------------------------------------------------------------------------------------
| Categories           | City Info (2-4 cols),          | Gov, Storage, Trade,          | Spans calculated dynamically
|                      | Resources (0-5 cols)           | Reducers, Enhancers,          |
|                      |                                | Military, Culture, Special    |
------------------------------------------------------------------------------------------------------------------------
| Special Features     | Number formatting with         | Palace+Governor merged        | Clickable city names for
|                      | .toLocaleString()              | into single column            | navigation
------------------------------------------------------------------------------------------------------------------------
| Control Elements     | Minimize (top-left),           | Same                          | Sprite-based icons with
|                      | Toggle (top-right)             |                               | hover states
------------------------------------------------------------------------------------------------------------------------
| Total Row            | Shows sums, #faeac6 bg, bold | Empty cells, same             | Last row of each table
|                      |                                | header styling                |
========================================================================================================================

Key Differences:
- Resources: Focus on quantities, warnings, production rates
- Buildings: Focus on construction status, building presence/absence  
- Both: Responsive visibility, current city highlighting, unified control scheme

Color Codes:
- #fdf7dd   = Light cream background
- #faeac6   = Header background (darker cream)
- #DBBE8C   = Category header background (brown/tan)
- #FF000050 = Red tint for storage warnings
- #80404050 = Orange tint for construction
